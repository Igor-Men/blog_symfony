Blog test project
==========

Clone project
------------
`git clone git@github.com:Igor-Men/blog_symfony.git`  

Install composer  
-------------
`composer update`  
set connection to database

Create dataabase
--------------
`php bin/console doctrine:database:create`

Run migrations
-------------
`php bin/console doctrine:migrations:migrate`

Run fixtures
-------------
`php bin/console doctrine:fixtures:load`
